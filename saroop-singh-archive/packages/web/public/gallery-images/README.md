# Gallery media

This directory contains best-available family scans, newspaper crops, one
derived crop, and labelled legacy AI experiments. A filename ending in
`-original` means the recovered source-side file for that catalogue record; it
does not prove a camera negative, physical print, or first-generation scan.

Files ending in `-restoration-N.png` are speculative outputs recovered from an
older AI workflow. Their prompts and exact models are unknown, and they may
alter identity, clothing, text, or scenery. Use them only through the archive's
labelled comparison interface.
