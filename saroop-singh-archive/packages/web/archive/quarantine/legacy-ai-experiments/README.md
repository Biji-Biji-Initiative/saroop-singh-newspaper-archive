# Quarantined legacy AI experiments

These files were recovered from the earlier archive build but are deliberately
excluded from the public site. They materially reconstruct faces, clothing,
bodies, text, or scenery and must not be described as historical restoration.

They remain in repository history for audit and research only. Exact prompts,
models, input settings, consent records, and review decisions were not preserved.
Do not republish them without a documented curatorial review.
