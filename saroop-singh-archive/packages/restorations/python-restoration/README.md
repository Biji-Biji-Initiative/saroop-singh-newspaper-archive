# Archived Python restoration prototype

This implementation is retained for historical research only. It is not
deployed, supported, or connected to the archive website.

The active production restoration route is implemented in
`packages/web/src/app/api/restore/route.ts` and deployed with Coolify. Read
[ARCHIVED.md](ARCHIVED.md) before inspecting, adapting, or running any material
in this directory.

The former Vercel deployment manifest, CLI dependency, and deployment scripts
were intentionally removed. Do not provision secrets or point Airtable
automations at this prototype.
